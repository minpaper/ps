boj 6171 땅따먹기 D5
-

https://www.acmicpc.net/problem/6171

CHT 문제

- $W_i$ 로 정렬
- $dp[i] = min_{0\le j < i}(dp[j]+max_{j < k\le i}(H[k]) W[i])$
- $f(x) = min_{0\le j < i}(max_{j< k\le i}(H[k])x+dp[j])$
- $dp[i] = f(W[i])$

정렬 $O(NlogN)$

$max_{j < k \le i}(H[k])$ seg로 관리해서 $O(logN) \space N$번하면 $O(NlogN)$

$ \to O(NlogN)$
